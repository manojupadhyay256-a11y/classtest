# 📝 Question Import Template (Word Document Guide)

To import questions into the **Classtest** platform, your Word document should follow this structured format. The importer looks for specific keywords to identify question types and answers.

---

## 1. Multiple Choice Questions (MCQ)
**Format:**
- Question text on one line.
- Options prefixed with `A)`, `B)`, `C)`, `D)`.
- Correct answer indicated by `Answer: [Key]`.

**Example:**
What is the capital of France?
A) Berlin
B) London
C) Paris
D) Madrid
Answer: C

---

## 2. True / False Questions
**Format:**
- Question text.
- Followed by `Answer: True` or `Answer: False`.

**Example:**
The sun rises in the West.
Answer: False

---

## 3. Fill in the Blanks
**Format:**
- Question text with a blank like `____` or `[...]`.
- Followed by `Answer: [Correct Word]`.

**Example:**
The chemical symbol for water is ____.
Answer: H2O

---

## 4. Jumbled Words / Reordering
**Format:**
- The jumbled word or phrase.
- Followed by `Answer: [Correct Order]`.

**Example:**
O-M-P-U-T-E-R-C
Answer: COMPUTER

---

## 5. Match the Following
**Format:**
- Use the `->` symbol to match pairs.
- Each pair on a new line.

**Example:**
Match the countries with capitals:
India -> New Delhi
USA -> Washington DC
Japan -> Tokyo

---

## 🛠 Points to Remember:
1. **Separation:** Leave a blank line between different questions.
2. **Marks:** By default, each question is assigned 1 mark unless specified in the UI during import.
3. **File Format:** Save your document as `.docx` (Word 2007+).
4. **No Images:** Currently, images inside Word docs are not supported for bulk import. Please add them manually in the Question Builder.
